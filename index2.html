Inhaltsverzeichnis HTML MAC-Adressen

MAC-Adressen



  • Steht für Media Access Control
  • Ist 48Bit lang
  • Hexadezimal Zahlen
  • Von der Netztwerkkarte abhängig
  • Ersten 6 Stellen stehen für den Hersteller
  • Letzten 6 Stellen sind eine einzigartige ID
  • Bleibt gleich, da es eine Physische Adresse ist










Synonyme für MAC-Adressen

  • Physische Adresse
  • Physikalische Adresse
  • Hardware Adresse
  • Geräte Adresse



  • Ping von Computer zu Computer



  • Ping von Server an alle Computer



  • Ping vom Server an 2 oder mehr Computer
