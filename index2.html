Inhaltsverzeichnis HTML MAC-Adressen

MAC-Adressen



  • Steht für Media Access Control
  • Ist 48Bit lang
  • Hexadezimal Zahlen
  • Von der Netztwerkkarte abhängig
  • Ersten 6 Stellen stehen für den Hersteller
  • Letzten 6 Stellen sind eine einzigartige ID
  • Bleibt gleich, da es eine Physische Adresse ist










Synonyme für MAC-Adressen

  • Physische Adresse
  • Physikalische Adresse
  • Hardware Adresse
  • Geräte Adresse



Arten von MAC-Adressen

  • Ping von Computer zu Computer



  • Ping von Server an alle Computer



  • Ping vom Server an 2 oder mehr Computer

Schreibweisen von Mac Adressen



Praxis



  • Router behält alle verbundenen Geräte im Auge
  • Router entscheidet wo die Daten ankommen sollen
  • Weist jedem Gerät/MAC-Adresse eine Lokale IP zu
  • Es wird sichergestellt, dass die Daten beim richtigen Gerät ankommen
  • Lokale IP und öffentliche IP unterscheiden sich stark
  • Man brauch MAC und Private IP
